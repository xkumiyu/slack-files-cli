# Slack Files CLI

# Installation

```
go get github.com/xkumiyu/slack-files-cli
```

# Usage

```
USAGE:
   slack-files-cli [global options] command [command options] [arguments...]

COMMANDS:
     config, c  view/set config
     files, f   list/donwload/delete files
     help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
